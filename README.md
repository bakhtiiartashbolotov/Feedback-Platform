# Feedback Platform
## Install instruction
1. Clone repository
    >git clone https://github.com/bakhtiiartashbolotov/Feedback-Platform.git
2. Create environment
   ```
   virtualenv .env
   ```
3. Intall requirements
   ```
   pip3 install -r requirements.txt
   ```
4. Run
   ```
    python3 manage.py runserver
    ```
